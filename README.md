## GitHub Pages veröffentlichen

1. Neues GitHub-Repository erstellen, z.B. `th-leistungsnachweis-quiz`.
2. Alle Dateien aus diesem ZIP in das Repository hochladen.
3. In GitHub: Settings → Pages.
4. Source: `Deploy from a branch`.
5. Branch: `main`, Folder: `/root`.
6. Speichern.
7. Nach kurzer Zeit ist die App erreichbar unter:
   `https://DEIN-NAME.github.io/REPOSITORY-NAME/`

## Smartphone-Installation

### Android / Chrome
1. Link öffnen.
2. Drei-Punkte-Menü öffnen.
3. „Zum Startbildschirm hinzufügen“ oder „App installieren“ wählen.

### iPhone / Safari
1. Link in Safari öffnen.
2. Teilen-Symbol antippen.
3. „Zum Home-Bildschirm“ wählen.

## Hinweis
Die Lernstände werden nur lokal auf dem jeweiligen Gerät gespeichert.
Es gibt keine zentrale Datenbank und keine Benutzerkonten.
